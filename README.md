# dead-poets-quiz

#  Which Dead Poet character are you?

An interactive personality quiz inspired by *Dead Poets Society*. Answer a series of thoughtfully crafted questions to discover which iconic character best matches your personality. Featuring a vintage parchment-inspired interface, smooth animations, and beautifully illustrated character results.

##  Features

-  8 immersive personality questions
-  7 unique Dead Poets Society character outcomes
-  Aesthetic vintage paper-inspired UI
-  Animated backgrounds and interactive effects
-  Fully responsive design
-  Built using pure HTML, CSS, and JavaScript

##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- SVG Illustrations
- Google Fonts
